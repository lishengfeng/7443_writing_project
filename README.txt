=================
Usage on Overleaf

(1) Log into Overleaf.com
(2) Click “New Project” button
(3) Choose “Upload Project”, and drag and drop the acm-alternate-latex.zip file

=================
Usage on Linux:

(1) Compile and create a PDF file: 
    $ make
    You should see a "paper.pdf" in the current directory. 

(2) Clean the directory:
    $ make clean

(3) Edit the latex file:
    $ vi main.tex
